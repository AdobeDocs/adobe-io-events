# Admin Console Audit Logs Changelog

Track released changes to the events and payloads.

The **Version** column tracks to the field `metadata.version` in every event payload (this is
usually the same as the release date of the changes). 

Note that this table only captures significant changes (e.g., new events) - we may not document
internal only changes here. As a result, the version at the top of the table may not
be the one that is currently active.
